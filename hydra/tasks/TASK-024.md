## Description

Create the source text for `GenerateMapAttribute` that the generator will inject into consuming projects. The attribute itself is NOT compiled into the generator assembly ŌĆö it is emitted as source text by the generator's `Initialize` method via `RegisterPostInitializationOutput`.

### Source Text

```csharp
// File: GenerateMapAttributeSource.cs (in generator project)
namespace DeltaMapper.SourceGen;

internal static class GenerateMapAttributeSource
{
    public const string AttributeName = "DeltaMapper.GenerateMapAttribute";
    public const string HintName = "GenerateMapAttribute.g.cs";
    public const string Source = """
        // <auto-generated />
        #nullable enable

        namespace DeltaMapper
        {
            [System.AttributeUsage(System.AttributeTargets.Class, AllowMultiple = true)]
            public sealed class GenerateMapAttribute : System.Attribute
            {
                public System.Type Source { get; }
                public System.Type Destination { get; }

                public GenerateMapAttribute(System.Type source, System.Type destination)
                {
                    Source = source;
                    Destination = destination;
                }
            }
        }
        """;
}
```

Key design decisions:
- Fully qualified `System.` references ŌĆö no `using` statements needed in generated source
- `AllowMultiple = true` ŌĆö a profile can declare multiple mappings
- Properties expose Source and Destination for compile-time reflection
- The attribute lives in the `DeltaMapper` namespace so users write `[GenerateMap(typeof(A), typeof(B))]` without extra usings

## Pattern Reference
