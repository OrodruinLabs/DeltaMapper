## Description

Implement the core `MapperGenerator : IIncrementalGenerator` that:

1. **RegisterPostInitializationOutput**: Injects `GenerateMapAttribute` source text (from TASK-024)
2. **Syntax Provider**: Filters for class declarations with `[GenerateMap(...)]` attribute on `partial class` types
3. **Transform**: For each `[GenerateMap(typeof(Src), typeof(Dst))]`:
   - Extract source and destination type symbols from the attribute arguments
   - Match destination writable properties to source readable properties by name (case-insensitive)
   - For directly-assignable types: emit `dst.PropName = src.PropName;`
4. **Output**: Emit a partial class file with a static method containing the assignment code

### Generated Code Shape (flat types only in this task)

```csharp
// <auto-generated />
#nullable enable

namespace UserProject
{
    public partial class UserProfile
    {
        private static void Map_User_To_UserDto(User src, UserDto dst)
        {
            dst.Id = src.Id;
            dst.Name = src.Name;
            dst.Email = src.Email;
        }
    }
}
```

NOTE: The `[ModuleInitializer]` registration with `GeneratedMapRegistry` is handled in TASK-027. This task focuses on the core syntax-provider pipeline and flat property assignment emission.
